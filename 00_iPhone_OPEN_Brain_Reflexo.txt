【iPhone】Open Brain_Reflexo.html

1. Unzip this folder in the Files app
2. Long-press Brain_Reflexo.html → Share → Open in Safari
3. Tap Play to start

Do NOT use index.html on iPhone (OS blocks separate JS/CSS files).

If "Open in Safari" is missing from Share:
→ Use the Web URL below (most reliable, no unzip needed)
  https://yosie-lab.github.io/brain-reflexo/
Or: Share → Edit/More → enable Safari, then try again
Or: Open with Chrome / Edge / Firefox from the Share sheet
