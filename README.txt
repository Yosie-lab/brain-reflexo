脳リフレクソ (Brain Reflexo) - 取扱説明書 (ReadMe)

この度は「脳リフレクソ」をご購入・ダウンロードいただき、誠にありがとうございます。
本作品は、心地よい音と光の泡をタップして弾けさせ、考えすぎた頭をサラッとクリアにするための脳疲労リセット空間（Webアプリケーション）です。

■ iPhone での起動方法（重要）
---------------------------------------------------------
iPhoneのセキュリティ制限により、Zip内の index.html から別ファイル（app.js 等）を
読み込むことができません。必ず次の手順で起動してください。

1. 「ファイル」アプリで brain_reflexo.zip をタップして解凍する
2. 解凍したフォルダ内の 「Brain_Reflexo.html」 を長押しする
3. 「共有」→「Safariで開く」を選ぶ
   ※ ファイルをタップしただけのプレビュー画面では、機種・OSによっては動きません
4. スタート画面で「Play」などをタップして開始

【「共有」→「Safariで開く」を選べない場合】
機種や iOS のバージョンによっては、共有メニューに「Safariで開く」が出ないことがあります。
そのときは次の順でお願いします（上ほど確実です）。

1. Web版URLを使う（いちばん確実・解凍不要）
   Safari のアドレス欄に次のURLを入力して開いてください。
   👇購入者様限定Web版URL
   https://yosie-lab.github.io/brain-reflexo/

2. 共有メニューに Safari を出す
   「Brain_Reflexo.html」を長押し →「共有」→ アイコン列を右端までスワイプ
   →「編集」または「その他」→ Safari をオン（お気に入りに追加）
   → もう一度「共有」から Safari を選ぶ

3. ほかのブラウザで開く
   共有シートに Chrome / Edge / Firefox などがあれば、それで開いても動作します。

上記でも難しい場合は、迷わず Web版URL をご利用ください。

■ PC / Android での起動方法
---------------------------------------------------------
お使いのPCやAndroidのブラウザで「index.html」または「Brain_Reflexo.html」を開いてください。
・PCの場合：
  ファイルをダブルクリックするか、ブラウザにドラッグ＆ドロップしてください。
・Androidの場合：
  ファイルマネージャーから HTML を開き、Chrome等で表示してください。

対応ブラウザ: Chrome / Safari / Microsoft Edge / Firefox（最新版推奨）

■ 音が鳴らないときの対処法 (Troubleshooting Sound)
---------------------------------------------------------
1. 消音（マナー）モードの解除（特にiOS端末）
   iPhone等の側面の消音スイッチがON（オレンジ色が見える状態）になっていると、ブラウザの音が消音されます。スイッチをOFFにしてください。
2. 端末・ブラウザの音量確認
   端末自体の音量が下がっていないか、またアプリ内設定（画面右上 ⚙️ アイコン）の音量スライダーが0%になっていないかご確認ください。
3. 画面のタップ
   ブラウザのセキュリティ仕様により、画面を一度タップするまで音が再生されない場合があります。
4. ページの再読み込み
   改善しない場合は、ブラウザの更新ボタンでページを再読み込みしてください。
