脳リフレクソ (Brain Reflexo) - 取扱説明書 (ReadMe)

この度は「脳リフレクソ」をご購入・ダウンロードいただき、誠にありがとうございます。
本作品は、心地よい音と光の泡をタップして弾けさせ、考えすぎた頭をサラッとクリアにするための脳疲労リセット空間（Webアプリケーション）です。

---------------------------------------------------------
■ パッケージ内容 (Files Included)
---------------------------------------------------------
- index.html   : アプリケーション本体（ダブルクリックで起動します）
- style.css    : スタイルシート
- app.js       : プログラム本体
- README.txt   : この説明書（本書）

---------------------------------------------------------
■ 起動方法 (How to Play)
---------------------------------------------------------
お使いのPCやスマートフォン、タブレットのブラウザで「index.html」を開くだけで、すぐに起動します。
・PCの場合：
  「index.html」ファイルをダブルクリックするか、お使いのブラウザ（Google Chrome, Safari, Microsoft Edge, Firefox 等）にドラッグ＆ドロップしてください。
・スマートフォンの場合：
  ファイルを端末に転送するか、ローカルファイル表示アプリ等を用いてブラウザで開いてください。

---------------------------------------------------------
■ 音が鳴らないときの対処法 (Troubleshooting Sound)
---------------------------------------------------------
1. 消音（マナー）モードの解除（特にiOS端末）
   iPhone等の側面の消音スイッチがON（オレンジ色が見える状態）になっていると、ブラウザの音が消音されます。スイッチをOFFにしてください。
2. 端末・ブラウザの音量確認
   端末自体の音量が下がっていないか、またアプリ内設定（画面右上 ⚙️ アイコン）の音量スライダーが0%になっていないかご確認ください。
3. 画面のタップ
   ブラウザのセキュリティ仕様により、画面を一度タップするまで音が再生されない場合があります。
4. ページの再読み込み
   改善しない場合は、ブラウザの更新ボタンでページを再読み込みしてください。
